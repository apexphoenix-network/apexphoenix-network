1. Executive Summary

Apex Phoenix Network (APXC) introduces a next-generation lightweight decentralized framework designed to eliminate the barrier of energy-intensive mining hardware. By utilizing WebAssembly (WASM) and high-throughput Seed Node WebSocket consensus, APXC empowers every smartphone, tablet, and PC browser to operate as a functional Light Mining Node.

The core mission of the protocol is to build a fair, energy-efficient, and censorship-resistant infrastructure capable of delivering real-time block telemetry and decentralized liquidity integration across Web3 ecosystems.

2. Network Architecture & Consensus

The protocol employs a hybrid consensus framework combining Proof-of-Capacity Light Mining (PoCLM) at the client side with dedicated VPS Seed Nodes serving as primary ledger anchors.

Target Block Time: 60 Seconds per Block
Network Engine: Python SQLite High-Concurrency Pipeline
Data Pipeline: REST API / CORS / WebSocket Bridge
Persistence Storage: Append-Only Time-Series Candle Ledger

Seed Nodes maintain time-series OHLC data accuracy, preventing chain forks and guaranteeing sub-second response times for global Light Node queries across DApp frontends.
